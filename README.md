# Portfolio Miguel Fernández Santos

Portfolio web profesional de Miguel Fernández Santos, periodista y redactor especializado en comunicación digital.

## 🎯 Características

- **Diseño responsivo**: Totalmente adaptado a dispositivos móviles, tablets y escritorio
- **Foto de perfil personalizable**: Sección interactiva para agregar tu foto en la navbar
- **Banner dinámico**: Carga fondos personalizados para el hero section
- **Paleta de colores armoniosa**: Degradación visual en tonos azules para profesionalidad
- **Navegación fluida**: Scroll suave y enlaces inteligentes
- **Enlaces funcionales**: 
  - Email abre Gmail directamente
  - Teléfono con protocolo `tel:`
  - Ubicación abre Google Maps
- **Secciones completas**:
  - Inicio (Hero)
  - Sobre mí
  - Experiencia Profesional
  - Formación Académica
  - Certificaciones
  - Idiomas y Habilidades
  - Datos de interés
  - Contacto

## 🚀 Tecnologías

- **HTML5**: Semántico y accesible
- **CSS3**: Variables personalizadas, responsive mobile-first
- **JavaScript vanilla**: Sin dependencias externas
- **Almacenamiento local**: LocalStorage para persistencia de foto de perfil y fondo

## 📋 Requisitos

- Navegador moderno (Chrome, Firefox, Safari, Edge)
- No requiere instalación ni dependencias

## 💻 Instalación Local

```bash
# 1. Clonar el repositorio
git clone https://github.com/miguelfernandezsantos/portfolio-miguel.git

# 2. Navegar a la carpeta
cd portfolio-miguel

# 3. Abrir en navegador
# Opción 1: Doble click en index.html
# Opción 2: Con servidor local (Python)
python -m http.server 8000
# Luego abrir en navegador: http://localhost:8000
```

## 📸 Uso

### Agregar foto de perfil
1. Haz click en el círculo con icono 📷 en la navbar
2. Selecciona una imagen
3. Se guardará automáticamente en el navegador

### Cambiar fondo del banner
1. Haz click en el botón 🖼️ en la esquina superior derecha del hero
2. Selecciona una imagen
3. Se guardará automáticamente en el navegador

### Contacto
- **Email**: Click abre Gmail Compose
- **Teléfono**: Click abre app de llamadas (en móvil)
- **Ubicación**: Click abre Google Maps
